# ✨ Lorexa – AI-Powered Conversational Learning App

> Transforming learning into an interactive conversation experience.

---

## 🚀 1. Project Overview
Lorexa is an AI-powered mobile application that enables users to engage in immersive conversations with digital avatars of historical figures, scientists, philosophers, and fictional characters.  

Instead of passive learning, Lorexa allows users to **ask, explore, and interact**, making knowledge more engaging and memorable.

---

## 🎯 2. Problem Statement
Most educational platforms rely on static content such as textbooks, videos, and articles. While informative, these methods often lack engagement and interactivity.

Lorexa addresses this gap by introducing:
- Real-time AI conversations  
- Personality-based responses  
- Interactive and immersive learning

## ⚙️ 7. Setup & Run Instructions

Follow these steps to run the project on a clean system:

### 1️⃣ Clone Repository
git clone https://github.com/rishi210421/lorexa.git

### 2️⃣ Open in Android Studio
- Open Android Studio  
- Select "Open Project"  
- Choose the cloned folder  

### 3️⃣ Sync Gradle
- Allow dependencies to install  

### 4️⃣ Configure Firebase
- Download `google-services.json`  
- Place it in `/app` folder  

### 5️⃣ Add API Keys Securely
- Use `local.properties`  
- Do NOT push keys to GitHub  

### 6️⃣ Run the App
- Use emulator or physical device  
- Click **Run ▶️**

## 🧱 10. Architecture Overview

Lorexa follows a **client–server architecture**:

- 📱 Android app → UI & user interaction  
- ☁️ Firebase → Authentication & database  
- 🤖 AI APIs → Conversational response generation
